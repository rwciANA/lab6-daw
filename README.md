# Lab 06 - Base de Datos

## Descripción

Se desarrolló un sistema de biblioteca que permite gestionar usuarios, libros y préstamos.

## Tecnologías utilizadas

* PostgreSQL
* Supabase

## Modelo de Base de Datos

Tablas implementadas:

* users
* books
* loans

Relaciones:

* Un usuario puede tener muchos préstamos
* Un libro puede estar en muchos préstamos

## Implementación

La base de datos fue implementada en PostgreSQL utilizando Supabase como Backend as a Service. Se crearon las tablas mediante sentencias SQL en el SQL Editor.

## Ejecución

1. Ingresar a Supabase
2. Abrir SQL Editor
3. Ejecutar el script SQL

## Evidencia

Se verificó la creación de tablas en el Table Editor y el correcto funcionamiento mediante consultas SQL.
